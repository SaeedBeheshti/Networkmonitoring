# Network Monitor (PyQt5)

This project is a **real-time network monitor** built with Python and PyQt5 that displays download and upload speeds graphically.

---

## Features

- Real-time display of **network download and upload speeds** (Bytes/s)  
- Graphical chart using **Matplotlib** integrated into a PyQt5 interface  
- Start monitoring with a single button  
- Keeps only the last 20 data points to maintain performance  

---

## Requirements

Make sure the following packages are installed:

- Python 3.6 or higher
- PyQt5
- Matplotlib
- psutil

### Install dependencies:

```bash
pip install pyqt5 matplotlib psutil
